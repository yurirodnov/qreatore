# Qreatore

A simple QR code generator built with **React**, **TypeScript**, and **Canvas API**.

## ✨ Key Features

- **Smart URL validation:** Uses native `URL.canParse()` API with detailed error messages
- **Multiple export formats:** Download QR codes as PNG, JPEG, or SVG
- **Clean UX:** No layout shifts, smooth transitions, inline validation
- **Hand-written code:** No AI-generated logic, only this README 🤖
- **TypeScript throughout:** Full type safety with discriminated unions for validation
- **Smooth design:** Made with help of [Colorhunt](https://colorhunt.co/)

## 🛠️ Local Development

Interested in the code? Wanna run it locally?

**Prerequisites:**

- [Node.js](https://nodejs.org/en) (v18 or higher) installed on your machine.

1.  **Clone the repo:**

    ```bash
    git clone https://github.com/yurirodnov/qreatore.git
    ```

    or via SSH

    ```bash
    git clone git@github.com:yurirodnov/qreatore.git
    ```

2.  **Go to a app directory:**
    ```bash
    cd qreatore
    ```
3.  **Install dependencies:**
    ```bash
    npm install
    ```
4.  **Run the dev server:**
    ```bash
    npm run dev
    ```
5.  **Open the link shown in the terminal (usually `http://localhost:5173`)**.

6.  **Build for production (optional)**:
    ```bash
    npm run build
    npm run preview
    ```
